## Never use these words

These words are vague, inflated, or associated with AI-generated marketing copy. Remove them from any documentation you write or review.

<Note>
These words are banned in all documentation. No exceptions.
</Note>

| Banned word | Why it's banned |
|---|---|
| Foster | Vague. Say what the product actually does. |
| Revolutionize | Marketing language. Documentation describes, not sells. |
| Landscape | Meaningless filler. |
| Seamless | Overused. If it were seamless, users wouldn't need documentation. |
| Robust | Vague. Be specific about what it handles. |
| Streamline | Overused. Say what specifically becomes faster or simpler. |
| Delve | AI-associated. Use "explore," "read," or "review." |
| Elevate | Marketing language. |
| Facilitate | Use "help" or the specific verb. |
| Groundbreaking | Marketing language. |
| Comprehensive | Overused. If it's comprehensive, the content proves it. |

## Never use these phrases

| Banned phrase | Alternative |
|---|---|
| Dives deep / Deep dive | "Covers in detail" or name the specific topic |
| This is where X comes in | Rewrite the sentence to state X directly |
| X is not just about; it's about | State what X does directly |
| X isn't just nice to have; it's... | State why X is required directly |
| You're not alone | Remove |
| Thrive in a fast-paced environment | Remove |
| The following (as a vague introducer) | Name what follows specifically |
| Click here | Write descriptive link text |
| See X for Y (as a lazy signpost) | Use a verb that describes the action: "read," "browse," "open" |

## Use sparingly

Use each of these a maximum of two to three times per page. Overuse makes them lose their meaning.

- Ensure
- Enhance
- Significant / significantly
- Effective / effectively
- Essential

## No mechanical transitions

Never use these to open a paragraph or sentence.

- Additionally
- Furthermore
- Moreover
- In conclusion
- To summarise / In summary
- As mentioned above
- As previously stated

These transitions add no meaning and make documentation sound like an essay draft.
